What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of objects, allowing developers to interact with and manipulate the content, structure, and styles of a website programmatically using JavaScript.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript by leveraging the DOM API methods. These methods are provided by the document object, and they allow you to target elements in various ways, such as by ID, class, tag name, or CSS selectors. Such as:
document.getElementById();
document.getElementsByClassName();
document.getElementsByTagName();
document.querySelector();
document.querySelectorAll();

What is the event delegation in the context of the DOM and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to manage events on its child elements. This works because events bubble up from the child to the parent (event propagation).

It is useful because-
1. Reduces the number of event listeners, improving performance.
2. Dynamically handles child elements, even if they are added after the event listener is set.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, use methods like setAttribute to add or update, getAttribute to retrieve, and removeAttribute to delete attributes. For styles, use the style property to directly set CSS, like element.style.color = 'red'; or element.style.backgroundColor = 'blue';.
